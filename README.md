## ✨ 功能亮点

| 功能             | 描述                             |
| ---------------- | -------------------------------- |
| 🖥️ **桌面小部件** | 待办事项直接显示在桌面，一眼看全 |
| 📋 **任务管理**   | 添加、编辑、完成、删除，一气呵成 |
| 🏷️ **分类管理**   | 工作、学习、生活，井井有条       |
| 📊 **数据统计**   | 日/周/月/年，看见你的坚持        |
| ⌨️ **快捷键**     | 按 \` 键，随时隐藏/显示小部件    |
| 🌙 **深色主题**   | 护眼设计，夜猫子最爱             |
| 🔄 **自动更新**   | 有新版本自动提醒，一键升级       |

## 💡 使用技巧

| 操作 | 说明 |
|------|------|
| 按 **`** 键 | 快速显示/隐藏桌面小部件（ESC 下方） |
| 双击托盘图标 | 打开主界面 |
| 右键托盘图标 | 快速退出 |
| 输入 "明天"、"下周一" | 智能识别日期 |
