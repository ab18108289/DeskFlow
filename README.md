## ✨ 功能亮点

| 功能             | 描述                             |
| ---------------- | -------------------------------- |
| 🖥️ **桌面小部件** | 待办事项直接显示在桌面，一眼看全 |
| 📋 **任务管理**   | 添加、编辑、完成、删除，一气呵成 |
| 🏷️ **分类管理**   | 工作、学习、生活，井井有条       |
| 📊 **数据统计**   | 日/周/月/年，看见你的坚持        |
| ⌨️ **快捷键**     | 按 \` 键，随时隐藏/显示小部件    |
| 🌙 **深色主题**   | 护眼设计，夜猫子最爱             |
| 🔄 **自动更新**   | 有新版本自动提醒                 |

## 💡 使用技巧

- 按 **`** 键（键盘左上角，ESC下方）快速显示/隐藏桌面小部件
- 双击系统托盘图标打开主界面
- 右键托盘图标可以快速退出
